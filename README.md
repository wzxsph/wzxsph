## Local preview

这是一个无构建依赖的静态站点：

```bash
python3 -m http.server 4173
```

打开 `http://127.0.0.1:4173/`。

## Deployment

推送到 `main` 后，`.github/workflows/pages.yml` 会自动发布到 GitHub Pages。
